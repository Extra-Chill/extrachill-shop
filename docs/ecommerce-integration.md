# E-commerce Integration

## WooCommerce Integration
- Homepage override using `extrachill_template_homepage` filter
- Template system combining theme filters with WooCommerce overrides
- Comprehensive styling with responsive design
- Cart icon in site header via theme hooks

## Ad-Free License System
- Cross-domain license validation using user meta
- Purchase processing via WooCommerce order hooks
- Username validation in cart and checkout
- Auto-completion for license orders
- Integration with `is_user_ad_free()` from users plugin

## Product Features
- Raffle system with progress counters
- Custom product fields for community features
- Category navigation and grid layouts
- Cart validation and checkout customization

## Security
- Input sanitization and output escaping
- Prepared statements for database queries
- Capability checks for admin functions
- Multisite authentication for cross-domain sessions
